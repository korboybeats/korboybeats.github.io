A standalone system-wide parametric equalizer with a real-time spectrum analyzer.
Must purchase in-app to use

Screenshots

Fully Parametric EQ

  • Unlimited fully parametric bands with 9 filter types
  • Peak, Low/High Pass, Low/High Shelf, Band Pass, Notch, and All Pass
  • Precise control over frequency, Q factor, and gain
  • Low Pass and High Pass filters up to 48 dB/oct
  • Per-band bypass and left/right channel assignment for stereo shaping
  • Global preamp with clipping detection

Interactive EQ Graph

  • Full frequency response graph from 20 Hz to 20 kHz with a ±24 dB range
  • Drag nodes to adjust frequency and gain in real time
  • Pinch a node to adjust Q factor
  • Double-tap to add a new band or reset an existing one
  • Pinch the graph to zoom into specific frequency ranges
  • Long-press a node for additional options

Real-Time Spectrum Analyzer

  • GPU-accelerated visualizer rendered with Metal
  • Bars, Curve, and Heatmap modes
  • Customizable bar count, spacing, radius, line width, fill, and glow
  • Logarithmic or linear frequency scaling with adjustable range
  • Heatmap palettes with configurable dB range
  • Peak hold, decay speed, smoothing, frame rate cap, and mirror mode

Preset System

  • Save and load full EQ configurations as presets
  • Built-in factory presets plus unlimited user presets
  • Assign custom SF Symbol icons for easy identification
  • Rename, delete, and organize from the overlay panel
  • Long-press the preset button for quick management

Built-In Media Controls

  • Play, pause, and skip tracks without leaving the app
  • Monitor now-playing info while fine-tuning your EQ
